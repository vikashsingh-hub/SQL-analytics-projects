# Music Store Analysis (SQL)

## Project Overview
This project focuses on analyzing the Music Store database using SQL in PostgreSQL and pgAdmin4. The project examines the dataset and solves different analytical queries to understand customer behavior, sales trends, music preferences and overall business performance of the music store.

---

## Project Objective
The objective of this project is to analyze the Music Playlist database using SQL and examine the dataset to understand the business operations of the music store. By solving different queries using SQL, this project helps in understanding customer behavior, sales trends, popular artists and genres, and overall business performance.
